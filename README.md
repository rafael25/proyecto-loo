Proyecto Final: Lenguajes Orientados a Objetos
====================================================

Descripción:
----------------------------------------------------

Esta aplicación utiliza la tecnología de serialización de Java para generar una base de datos básica dedicada a almacenar libros, principalmente en formato pdf.

### Funciones

+ Mostrar todos los libros almacenados en una tabla
+ Agregar libros a la base de datos
+ Editar el libro seleccionado de la tabla
+ Extraer el libro almacenado
+ Eliminar el libro seleccionado
